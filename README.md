## to-mvp

A Claude skill that turns your own codebase into a self-paced course to level you up from frontend/creative developer to confident full-stack.
It inspects your project, maps the stack and architecture, then generates an ordered, junior-friendly curriculum — easy to advanced — where every lesson is anchored to your real code. The whole thing is taught in the Gordon Zhu "Watch and Code" style (read code before writing it, don't trust, verify, fundamentals before frameworks) and culminates in an MVP capstone that has you rebuild the project's core features end to end. Delivered as a downloadable course you can work through offline, at your own pace.

Install notes: `npx skills add victorlabigar/skills --skill to-mvp`
